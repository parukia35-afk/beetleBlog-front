# BeetleBlog (蟲生.蟲森) 🌿🐞
這是一個專為甲蟲愛好者設計的部落格平台。結合了我所學的專業背景與對鍬形蟲的熱愛，旨在解決甲蟲社群資訊碎片化的問題，提供一個結構化、美觀且易於編輯的知識分享空間。
---
## 🛠️ 技術棧 

| 類別 | 關鍵技術 | 說明 |
| :--- | :--- | :--- |
| **核心框架** | **Vue 3** | 使用 **Composition API** 進行開發。 |
| **構建工具** | **Vite 7** | 提供極速的開發環境與專案打包。 |
| **UI 框架** | **Vuetify 3** |  |
| **狀態管理** | **Pinia** | 搭配 **Persistedstate** 插件實現數據持久化（如登入狀態）。 |
| **編輯器** | **Tiptap** | 富文本編輯器，支援表格、圖片及自定義樣式。 |
| **路由管理** | **Vue Router** | 搭配 `unplugin-vue-router` 實現自動化路由配置。 |
| **網路請求** | **Axios** | 處理與後端 API 的異步通訊。 |
| **開發工具** | **ESLint / Sass** | 確保代碼規範與支援 **SCSS** 預處理。 |
